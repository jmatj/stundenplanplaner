## Projekt einrichten

1. Docker und Make installieren
2. Projekt klonen
3. Datei auth.cfg im Projektverzeichnis erstellen:

```
[authentication]
username=<hsr-username>
password=<hsr-passwort>
```

Die Demo kann nun mittels `sudo make demo` ausgeführt werden.
